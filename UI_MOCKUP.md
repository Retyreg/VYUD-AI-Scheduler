## Editable Features

### Column Types:
1. **Дата (Date)** - Date picker, format YYYY-MM-DD
2. **Platform** - Dropdown: LinkedIn, Telegram, Instagram, Facebook
3. **Текст поста** - Large text area, expandable
4. **Media Type** - Dropdown: image, video, carousel, none
5. **Media Description** - Text area
6. **URL** - Text input for Google Drive/S3 links

### User Interactions:
- ✏️ Click any cell to edit
- ➕ Click "+ Add row" to add new post
- ➖ Click row number and delete to remove
- 💾 Changes auto-save to session state
- 📋 Copy/paste works in cells
- ⌨️ Tab to move between cells

## Error States

### Missing API Key
```
❌ GEMINI_API_KEY не настроен. Пожалуйста, добавьте его в .streamlit/secrets.toml

See documentation: AI_STRATEGY_GUIDE.md
```

### Validation Error
```
❌ Пожалуйста, заполните обязательные поля: Тема/Продукт и Целевая аудитория
```

### API Error
```
❌ Ошибка при генерации плана: Invalid API key

Попробуйте еще раз или проверьте настройки API.
```

### JSON Parse Error
```
❌ Ошибка парсинга JSON: Unexpected token

Модель вернула невалидный JSON. Попробуйте еще раз.
```

## Mobile View Notes

The interface is responsive and will adapt to mobile:
- Form fields stack vertically
- Table becomes horizontally scrollable
- Buttons stack in single column
- Text areas adjust to screen width

- Primary: Blue (#1f77b4) - for primary buttons
